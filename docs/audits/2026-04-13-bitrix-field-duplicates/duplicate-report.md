## Ограничения

- Отчет использует common layout карточек (`scope=C`). Если у пользователей включены персональные настройки карточки, фактическое отображение может отличаться.
- Смысловой дубль определяется эвристически по названию, типу, enum/справочникам и расположению. Финальное решение об объединении требует бизнес-подтверждения.
- Поля с одинаковым technical code в разных воронках вынесены отдельно как переиспользование, а не как дубль.
