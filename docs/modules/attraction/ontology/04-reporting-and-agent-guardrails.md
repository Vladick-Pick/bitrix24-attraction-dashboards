# 04. Отчетность И Ограничения Для Агентов

Этот документ фиксирует, как отчеты, агенты и автоматизация обязаны читать
онтологию `Привлечения`.

Отчетность не является отдельным слоем онтологии. Это слой применения:
он берет определения, состояния и решения и переводит их в безопасные метрики,
дашборды, предупреждения о качестве данных и правила для агентов.
