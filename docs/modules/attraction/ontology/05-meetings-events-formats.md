# 05. Встречи, События И Форматы

## Встреча Как Объект

Встреча - отдельный объект, не равный стадии.

| Атрибут | Смысл |
| --- | --- |
| Дата и время | Когда встреча назначена или проведена. |
| Тип | Zoom, очная встреча, мероприятие, 1-1, встреча с основателем или другой тип. |
| Место | Zoom, офис, площадка события, внешняя площадка. |
| Куратор | Кто отвечает за организацию и качество встречи. |
| Статус | Назначена, проведена, перенесена, отменена, не состоялась. |
| Результат | Какой следующий шаг выбран после встречи. |

Для Привлечения штрих на этапе `Встреча-знакомство` приоритетны Zoom-встречи,
если нет отдельной причины вести человека в очный формат.

## Конверсионное Событие

Конверсионное событие - управляемый формат, который должен изменить состояние
потенциального участника и продвинуть его вправо:

```text
интерес -> понимание ценности -> осознание задачи -> ощущение fit -> оформление
```

Формат сам по себе не доказывает эффект. Доказательством является связка:

- человек был приглашен / подтвердил / пришел;
- формат имеет целевое состояние;
- после формата зафиксирован следующий шаг;
- участник действительно двинулся дальше по маршруту.

## Целевые Эффекты

| Эффект | Что означает |
| --- | --- |
| Активация | Первый значимый опыт и интерес к продолжению. |
| Демонстрация | Понимание, как работает клуб и в чем конкретная ценность. |
| Проблематизация | Осознание собственной задачи и связи клуба с этой задачей. |
| Адмиссия | Проверка и переживание взаимного соответствия. |

Один формат может иметь основной и дополнительный эффект. В отчетности нужен
главный эффект события, иначе формат начнет считаться сразу всем.

Рабочий каталог взят из таблицы `Конверсионные события`. Он фиксирует, какие

| Формат | Целевой эффект | Клуб / сегмент | Ответственный контур | Комментарий |
| --- | --- | --- | --- | --- |
| Гостевая встреча | Активация | СФ | ПАУ | Еженедельно, четверг 16:30. Для потенциального участника бесплатно. |
| Нетворкинг-сессия | Активация | СФФ | Клубные активности | Ежемесячный формат. |
| БКМ | Активация, демонстрация | СФ, СФФ | Клубные активности | Может давать и первый опыт, и демонстрацию устройства клуба. |
| Отраслевая конференция | Активация | СФ, СФФ | Не уточнено | Требует следующего шага после события. |
| Отраслевая встреча | Демонстрация | СФ, СФФ | Не уточнено | Показывает релевантную среду и людей. |
| Лайфстайл-мероприятия | Демонстрация | СФ, СФФ | Не уточнено | Демонстрация среды и качества сообщества. |
| Горячее кресло | Демонстрация | СФФ | Не уточнено | Показывает механику разбора и ценность участников. |
| Клубное мероприятие в гостиной | Демонстрация | СФ, СФФ | Не уточнено | Требует фиксации следующего шага. |
| 1-1 с продюсером | Проблематизация, адмиссия | СФФ | Привлечение | Сильный формат для проработки запроса и fit. |
| 1-1 с активным участником | Проблематизация | СФ, СФФ | ПАУ | Работает через peer proof и конкретизацию задачи. |
| 1-1 с основателями | Адмиссия | СФ | Привлечение | Формат для финальной ценностной сверки. |
| ОП `Построение сообществ` | Активация, демонстрация | СФ, СФФ | Не уточнено | Может быть форматом, если допроектирован под переход. |
| Бизнес-миссии | Активация | СФ, СФФ | Не уточнено | Важен post-event route. |
| Путешествия | Активация | СФ, СФФ | Не уточнено | Важен post-event route. |
| Личная встреча | Активация, демонстрация | СФ, СФФ | Не уточнено | Живой контакт может быть отдельным форматом. |
| Сопровождение на мероприятии | Демонстрация | СФ, СФФ | ПАУ | Помогает человеку увидеть формат изнутри, а не просто посетить событие. |

## Правила Для Событий

| Правило | Значение |
| --- | --- |
| Событие без следующего шага дефектно | После формата должен быть звонок, встреча, задача, адмиссия, контракт, возврат или проигрыш. |
| Посещение не равно прогрессу | Человек мог прийти, но не изменить состояние и не пойти дальше. |
| Партнерское лидогенерационное мероприятие не равно конверсионному событию | Оно становится частью привлечения только после контакта КИ и маршрута в `Встреча-знакомство` или другой следующий шаг. |
| Конверсионное событие может работать для разных этапов | Одно мероприятие остается событием Привлечения, если посещение связано с прямой сделкой Привлечения; отчет должен показывать, с каких этапов онтологии звали участников. |
| Эффект протухает | Активация и демонстрация теряют силу без быстрого follow-up. |
| Формат не равен состоянию | Один и тот же формат может работать по-разному для разных людей и сегментов. |
