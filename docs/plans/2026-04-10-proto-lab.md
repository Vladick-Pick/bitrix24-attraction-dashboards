# Prototype Lab Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a separate `apps/proto` frontend app for rapid design prototyping with local pinned comments that stay outside the main product.

**Architecture:** Add a new Vite React workspace app dedicated to design experiments. The app will render switchable prototype scenes, expose a comment mode for dropping pins on the canvas, and persist comments to a local JSON file through Vite dev/preview middleware so the notes remain machine-readable for future iterations.

**Tech Stack:** Vite, React 19, TypeScript, Tailwind CSS v4, local Vite middleware, Testing Library, Vitest
